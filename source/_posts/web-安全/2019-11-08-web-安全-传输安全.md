## Http 窃听

获取用户数据，可能包括用户名、密码

## Http 篡改

1. 插入广告
2. 重定向网站
3. 植入脚本
4. 无法防御 XSS 和 CRSF 攻击

## Https

通过 TLS(SSL) 加密传输数据，保证传输安全

通过证书确认服务器身份，防止中间人攻击

1. 证书不能伪造
2. 证书私钥不能揭露
3. 域名管理权不能泄露
4. CA 监管

**证书由 `CA` 提供**
